# PCManFM-Qt

The Qt port of the LXDE file manager PCManFM.

Includes libfm-qt, the qt port of the libfm-qt - a library providing components
to build desktop file managers.

Issue tracker:
  https://github.com/lxde/pcmanfm-qt/issues

LXQt website:
  http://lxqt.org

LXDE website:
  http://lxde.org
